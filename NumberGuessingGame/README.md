# 🎮 Java Number Guessing Game
## 📌 Project Overview
This is a Java console application where users guess a randomly generated number within a limited number of attempts. The game provides hints, scoring, difficulty levels, and a play-again option to enhance the user experience.
## 🚀 Features
✅ Difficulty Levels (Easy, Medium, Hard)
✅ Score System
✅ 5 Attempt Limit
✅ Random Number Generation
✅ Hint System (Too High / Too Low)
✅ Play Again Option
✅ Game Over and Winning Screens
## 🛠 Technologies Used
* Java
* VS Code
* GitHub
## ▶️ How to Run
1. Open the project in VS Code.
2. Compile the Java file.
3. Run the program.
4. Select a difficulty level.
5. Start guessing the secret number.
6. View your score and play again if desired.
