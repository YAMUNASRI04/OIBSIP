# Online Examination System
## Description
A Java console-based Online Examination System developed as part of the Oasis Infobyte Java Development Internship.
## Features
- User Login System
- Profile Update
- Password Management
- Multiple Choice Questions
- Automatic Evaluation
- Score Calculation
- Grade Generation
- Result Summary
- Logout Functionality
## Technologies Used
- Java
- Object-Oriented Programming
- Collections
- Scanner Class
## Project Structure
- Main.java
- LoginSystem.java
- Dashboard.java
- ExamSystem.java
- ProfileUpdate.java
- Question.java
- User.java
## How to Run
1. Compile all files:
   javac *.java
2. Run:
   java Main
